🧠 HCF-4α Framework Validation

Analysis code for the HCF-4α framework validation study using NHANES data (2017-2023).

📌 For Peer Review
This repository contains all analysis scripts for the manuscript "H = R_carrier × R_consciousness: A Computable Theory of Psychophysiological Health".

📁 Repository Structure

  • Scripts 01-08: Data cleaning and preparation
  • Scripts 09-11: Paper 1 - HCF typology analyses
  • Scripts 12-15: Paper 2 - Pathway clustering and alpha factors
  • Scripts 16-20: Paper 3 - Validation and sensitivity analyses

📊 Key Findings

  • α₂ identified as transdiagnostic protective factor (ORs 0.39-0.82, NNT=21)
  • Perseveration Paradox: Highest depression (PHQ-9=11.3) with lowest inflammation (CRP=3.3 mg/L)
  • Compression Hypothesis: Fatigue ↑1.10, Suicidal ideation ↓0.40
  • Incremental value: Alpha factors explain +9.5% variance, 3× physiological measures
